Paquete listo para despliegue del Seminario de Teoría de Galois - SEMAC

Contenido:
- index.html: página estática autocontenida. Sube este archivo como raíz del sitio.

Opciones rápidas:
1. Netlify Drop: arrastra esta carpeta descomprimida a Netlify Drop.
2. GitHub Pages: sube index.html a un repositorio, activa Pages desde Settings > Pages.
3. Vercel: importa el repositorio o sube la carpeta desde la CLI.

Importante:
- El archivo principal debe llamarse index.html para que el sitio abra directamente en la URL raíz.
- No renombres index.html después de desplegar.
